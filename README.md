## 🛠️ Active Projects

### [Project Delta Pen (DP-1)](https://github.com/forgedlogicdev/project-delta-pen)
A high-precision, motorized coaxial TIG filler wire feeder powered by an RP2040 microcontroller. It features dynamic force-sensitive feedback (FSR strip) and high-speed safety interrupts.
* **Firmware**: CircuitPython (RP2040)
* **Mechanics**: Coaxial feed path, direct-drive V-groove pinch gear
* **Licensing**: Proprietary & Confidential
